# Cognitive Learning Hub (CLH) 🧠

A powerful Python Flask + MongoDB web application for syllabus mapping, knowledge tracking, and AI-powered study visualization.

## ✨ Features
1. **User Authentication & Syllabus Hub**: Register and select subjects (Physics, C Programming, etc.) to pre-load key topics.
2. **Confidence-Based Mastery**: Rate your confidence (1-5) for each topic.
3. **AI Visualization Dash**:
   - **Mind Maps**: Automatic network visualization of your knowledge base.
   - **Flowcharts**: Track the logical progression of topics.
   - **Flashcards**: Interactive study cards with AI-generated "Golden Points".
4. **Color Psychology Integration**: Topics change color from Red (Weak) to Green (Strong) based on your confidence score.
5. **Knowledge Decay Tracker**: Topics not reviewed for 7 days automatically downgrade in confidence (e.g., Green → Yellow).
6. **AI Summarization**: Integrates with SambaNova (DeepSeek-R1) to generate intelligent summaries.
7. **Responsive Dashboard**: Glassmorphism UI with real-time stats and revision alerts.

## 🚀 Deploy to Render (One Click)

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy)

### Steps:
1. **Fork** this repository to your GitHub account.
2. Go to [Render Dashboard](https://dashboard.render.com/) → **New Web Service** → Connect your GitHub repo.
3. Render will auto-detect `render.yaml`. Set the following environment variables:

| Variable | Description |
|---|---|
| `SECRET_KEY` | Auto-generated by Render (or set your own) |
| `MONGO_URI` | Your MongoDB Atlas connection string |
| `SAMBANOVA_API_KEY` | Your SambaNova API key |
| `HF_TOKEN` | Optional: HuggingFace token |
| `OPENAI_API_KEY` | Optional: OpenAI API key |

4. Click **Deploy** 🎉

## 🛠 Local Development

### Prerequisites
- Python 3.8+
- MongoDB (Local or Atlas)
- SambaNova AI API Key

### Installation
1. Clone the repo:
   ```bash
   git clone https://github.com/aswiiz/chlbot.git
   cd chlbot
   ```
2. Create virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Linux/Mac
   venv\Scripts\activate     # Windows
   ```
3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
4. Set up environment variables — copy `.env.example` to `.env` and fill in values:
   ```bash
   cp .env.example .env
   ```
5. Run the application:
   ```bash
   python app.py
   ```
6. Access the app at `http://localhost:5000`.

## 🛠 Tech Stack
- **Backend**: Flask, PyMongo, Flask-Login
- **Frontend**: HTML5, Vanilla CSS (Glassmorphism), JavaScript
- **Database**: MongoDB Atlas
- **AI**: SambaNova (DeepSeek-R1)
- **Deployment**: Render (Gunicorn)
